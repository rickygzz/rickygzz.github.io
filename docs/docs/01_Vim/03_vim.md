# Setting up shortcutsLines

| Shortkey            | Description    |
| ------------------- | -------------- |
| `yyp`               | Duplicate line |



## Exiting

| Command      | Description |
| ------------ | ----------- |
| `:q`         | Close file          |
| `:qa`        | Close all files     |
| `:w`         | Save                |
| `:wq` / `:x` | Save and close file |
| `ZZ`         | Save and quit       |
| `:q!` / `ZQ` | Quit without checking changes |
